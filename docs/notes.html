How it works

The interface talks directly to GitHub. Notes are appended to notes/notes.json and committed to the main branch.

  • Keep entries short—bullets render exactly as plain text in the log.
  • Avoid private data. Everything you publish is visible to the community.
  • Rotate personal access tokens often and grant repository contents scope only.
Quick tip: Store a personal access token in localStorage under SOLAR_GH_TOKEN. Submissions require this token to create commits on your behalf.

Publish a note

Publishing happens immediately using your stored GitHub token. Double-check sensitive information before clicking submit.
