How notes are stored

Notes live in notes/notes.json within this repository. Each package key holds an array of short bullet points so you can quickly scan what changed between installs.

  • Use concise sentences — the log renders as plain text for quick copy and paste.
  • Keep secrets out of the repository. Everything you publish becomes public immediately.
  • Rotate personal access tokens regularly and scope them to repository content.

Publish a note

Store a personal access token in localStorage as SOLAR_GH_TOKEN to enable submissions.

Submitting uses your browser and a personal GitHub token with repository write access. Review the note before publishing — the change becomes public immediately.
