Welcome to Solar Neo

Multi-backend helper (dnf5/Flatpak). Releases are hosted outside /docs under /Versions.

Install (CLI)

curl -L https://randompixle.github.io/Solar-Neo/Versions/Solar-Neo-v0.7.0/Release.zip -o solar-neo.zip
unzip solar-neo.zip -d Solar-Neo
cd Solar-Neo/Release
./install.sh
solar --version
